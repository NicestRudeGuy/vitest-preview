# Example with react-testing-library

## Instruction

- Install the dependencies:

```bash
pnpm install
```

- Open **Vitest Preview Dashboard**:

```bash
pnpm vitest-preview
```

- Run tests:

```bash
pnpm test
```

- Modify [App.test.ts](./src/App.test.tsx) to see the changes reflects in **Vitest Preview Dashboard** immediately.
